# Chinese Character Atlas — Roadmap

> Every character tells a story. Every story opens a civilization.

---

# Vision

Build the most complete interactive encyclopedia for learning Chinese through its language, writing system, history, literature, philosophy, and culture.

The project should remain useful from beginner level to advanced study.

---

# Phase 1 — Foundation

Goal:

Build the architecture and standards.

Deliverables:

- README
- CONSTITUTION
- ARCHITECTURE
- STYLE GUIDE
- WORD TEMPLATE
- COMPONENT TEMPLATE
- CHARACTER TEMPLATE
- LESSON TEMPLATE
- ROADMAP

Status:

🟢 In Progress

# Phase 2 — Core Vocabulary

Goal:

Create the first 500 most useful words.

Every entry should include:

✓ pronunciation

✓ examples

✓ history

✓ common mistakes

✓ modern usage

✓ cultural notes

✓ references

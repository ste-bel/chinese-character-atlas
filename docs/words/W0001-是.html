W0001 — 是

Pronunciation

是
Pinyin: shì
Meaning: to be; is; are; am; yes; correct

Core Usage

是 identifies what something is.

Pattern:

A 是 B
A is B

Example

我是加拿大人。
Wǒ shì Jiānádà rén.
I am Canadian.
Literal: I / am / Canada / person.

Question

你是加拿大人吗？
Nǐ shì Jiānádà rén ma?
Are you Canadian?

✅ 是，我是。
Shì, wǒ shì.
Yes, I am.

❌ 不是，我不是。
Bú shì, wǒ bú shì.
No, I am not.

Common Mistake

Chinese does not normally use 是 before simple adjectives.

✅ 我很好。
Wǒ hěn hǎo.
I am good.

❌ 我是很好。
Unnatural in beginner Mandarin.

Atlas Note

English uses “to be” everywhere. Chinese uses 是 more selectively. Learning when not to use 是 is just as important as learning when to use it.

# W0008 — 有

## Pronunciation

**有**
**Pinyin:** yǒu
**Tone:** Third tone (falling-rising)
**Meaning:** to have; there is; there are; to exist

## Core Usage

有 has two closely related uses:

**Function 1 — Possession:**
Subject + 有 + Object → Subject has Object

**Function 2 — Existence:**
Location + 有 + Object → There is/are Object at Location

## Examples — Possession

{{audio:我有一个问题。}}
我有一个问题。
*Wǒ yǒu yīgè wèntí.*
I have a question.
Literal: I / have / one / [measure word] / question.

{{audio:她有两个孩子。}}
她有两个孩子。
*Tā yǒu liǎng gè háizi.*
She has two children.
Literal: She / have / two / [measure word] / child.

## Examples — Existence

{{audio:桌子上有一本书。}}
桌子上有一本书。
*Zhuōzi shàng yǒu yī běn shū.*
There is a book on the table.
Literal: Table / on / have / one / [measure word] / book.

{{audio:这里有人吗？}}
这里有人吗？
*Zhèlǐ yǒu rén ma?*
Is anyone here?
Literal: Here / have / person?

## Question

{{audio:你有时间吗？}}
你有时间吗？
*Nǐ yǒu shíjiān ma?*
Do you have time?

✅ {{audio:有，我有时间。}}
有，我有时间。
*Yǒu, wǒ yǒu shíjiān.*
Yes, I have time.

❌ {{audio:没有，我没有时间。}}
没有，我没有时间。
*Méiyǒu, wǒ méiyǒu shíjiān.*
No, I don't have time.

## Negation: 没有 not 不有

有 is the only common verb in Mandarin that uses 没 for negation, not 不.

✅ 我没有时间 (I don't have time)
❌ 我不有时间 (wrong)

## Common Mistake

English speakers use 有 where Chinese uses 是:

❌ 我有加拿大人 (wrong)
✅ 我是加拿大人 (I am Canadian — use 是 for identity)

有 is for possession and existence, not identity.

## Atlas Note

{{atlas-note}}
有 and 没有 together form the most common way to say yes and no in response to "do you have…" questions. 有 (yes) and 没有 (no/none) function almost like standalone affirmative and negative words in everyday speech.
{{/atlas-note}}

# AI Collaboration Guide

Welcome to the Chinese Character Atlas.

This repository is maintained by humans and AI assistants working together.

The repository — not any individual AI — is the project's permanent memory.

---

## Start here

Before making any change, every AI assistant must read:

1. `AI_RULES.md` — universal rules that apply to all contributors (AI or human)
2. `CONSTITUTION.md` — project mission and principles
3. `ARCHITECTURE.md` — directory layout and technical decisions
4. `EDITORIAL_POLICY.md` — accuracy and sourcing standards
5. `STYLE_GUIDE.md` — writing style and Markdown syntax extensions
6. `SCHEMA.md` — entity types and relationship model
7. The relevant template for the entity type you are working on

---

## Model-specific guidance

Soft role suggestions are in the individual files below.
These are guidelines, not constraints. Any model may work on any area.
Quality and accuracy matter more than role assignment.

- `CLAUDE.md` — engineering, build tooling, automation
- `CHATGPT.md` — research, content, Chinese explanations
- `COPILOT.md` — code completion, small edits
- `GEMINI.md` — source comparison, scholarly summaries

**The universal rules in `AI_RULES.md` take precedence over everything in these files.**

---

## Core principle

> The Atlas is more important than the AI.
> Models will change. Knowledge remains.

# AI Collaboration Guide

Welcome to the Chinese Character Atlas.

This repository is designed to be maintained by humans and AI working together.

The repository—not any individual AI—is the project's permanent memory.

Every AI should read these files before making changes:

1. CONSTITUTION.md
2. ARCHITECTURE.md
3. EDITORIAL_POLICY.md
4. STYLE_GUIDE.md
5. SCHEMA.md
6. WORD_TEMPLATE.md
7. COMPONENT_TEMPLATE.md
8. ROADMAP.md

The objective is to improve the Atlas.

Never optimize the repository for one specific AI.

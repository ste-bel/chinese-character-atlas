# Contributing to the Chinese Character Atlas

Thank you for contributing.

Our goal is quality over quantity.

## Before adding new content

Ask yourself:

- Is it accurate?
- Is it useful?
- Is it historically honest?
- Is it understandable to a learner?
- Does it connect to existing knowledge?

## General Rules

- Never invent facts.
- Verify quotations.
- Explain uncertainty.
- Prefer primary sources whenever possible.
- Link related pages instead of repeating information.

## New Word Checklist

Every new word should eventually include:

- Meaning
- Pronunciation
- Examples
- Question and answers
- New words introduced
- Character construction
- Historical evolution
- Cultural significance
- Modern usage
- Regional differences
- Common mistakes
- Mnemonics
- References
- Quality checklist

## New Character Checklist

Every character should explain:

- Origin
- Evolution
- Components
- Meaning
- Pronunciation
- Related words
- Cultural notes
